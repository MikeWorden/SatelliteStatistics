# SatelliteStatistics
Exploration of Satellite launches 1990 - 2022

Dash/Plotly application exploring launches, satellite missions, and countries of origin for launches 1991 - 2022
Leverages the UCS Satellite Database - An in-depth details on the 7,560 satellites currently orbiting Earth, including their country of origin, purpose, and other operational details.

To use:

1) Clone the repo to a local folder

2) Create a virtual environment with at-leas Python 3.11

3) Install necessary libraries (pip install -r requirements.txt)

4) From the folder, run ucsplot.py

5) Navigate to http://127.0.0.1:8089/

6) Application includes a number of interactions including dropdowns, and free-text search to filter down to meaningful results
